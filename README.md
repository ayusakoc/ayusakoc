# About Me
I am a Software Engineering student at Ankara Science University, specializing in **Artificial Intelligence**. My work primarily focuses on high-precision **Medical Image Segmentation** (Liver, Tumor, and Vessels) using state-of-the-art architectures and building scalable **RAG (Retrieval-Augmented Generation)** pipelines.

- 🔬 Currently optimizing **nnU-Net** and **Mamba** based architectures for vessel segmentation.
- 🤖 Developing RAG-based chatbots using **FastAPI**, **ChromaDB**, and **Ollama**.
- 🛠️ Deeply interested in **Object-Oriented Programming** and efficient AI deployment via **Docker**.
